Statistics Calculator!

Write the web app to prompt accepting for positive integers until the user enters a negative interger

  • If the entered text is not a positive integer number, then the web app will keep asking for a number until the valid positive interger is entered
  • If entered text is a negative integer,then the web app will show the satatistics (average, min, max) of all entered positive integers

Note:

  1. You need to implement functions to check whether an entered text is a positive integer or not and whether an entered text is a negative interger or not
  2. You need to implement functions to compute average, min, and max of list of integers
